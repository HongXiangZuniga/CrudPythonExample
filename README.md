# CrudPythonExample
It is a example to Crud Api in python with flask and connected to mongodb.

ENV:
```
PORT=
MONGOURI=
```

The struct of mongo data is:

```json
{
  "id":22,
  "name":"Hong Xiang",
  "email":"hongxiang17@gmail.com",
  "age":27,
  "entryDate":{"$date":"2018-06-25T21:46:44.000Z"},
  "country":"Chile"
}
 ```
Endpoints:
```
/user/id/:id              
/user/field/:value
```

Run:
```
Make run
```
