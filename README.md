# CrudPythonExample

 
La estructura utilizada en el mongo es:

```json
{
  "id":22,
  "name":"Hong Xiang",
  "email":"hongxiang17@gmail.com",
  "age":27,
  "entryDate":{"$date":"2018-06-25T21:46:44.000Z"},
  "country":"Chile"
}
 ```
 
Los endpoint:
```
/user/id/:id              
/user/country/:country
```
